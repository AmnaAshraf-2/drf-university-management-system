# University Management System (Django DRF)

This is a backend API project built using **Django Rest Framework** for managing
students, teachers, departments, courses, attendance, and assessments.

## Features
- User authentication (Students & Teachers)
- Role-based access
- Departments & Courses management
- Enrollment system
- Attendance & assessments APIs

## Tech Stack
- Python
- Django
- Django Rest Framework
- SQLite (development)

## Setup Instructions

1. Clone the repository
2. Create virtual environment
3. Install dependencies:
   pip install -r requirements.txt
4. Run migrations:
   python manage.py migrate
5. Start server:
   python manage.py runserver
